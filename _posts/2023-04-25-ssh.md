## SSH Info

### Allow root to SSH

* Edit the following file: `etc/ssh/sshd_config` and _comment out_ the following line
```terminal
PermitRootLogin without-password
```
* Just below it, add the following line
```terminal
PermitRootLogin yes
```
* Then restart SSH
```terminal
service ssh restart
```
