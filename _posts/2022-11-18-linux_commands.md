## Simple Linux Commands

### Copy

```terminal
sudo cp -r /path/to/directory /path/to/location/new-name
```

> -r means recursive and should be used to perform action on all sub folders and files
{: .prompt-tip }

### Remove

```terminal
sudo rm -r /path/to/dir
```
